# Derived registry files

This directory contains combined metadata files derived from the resource
definitions in the [`resource/`](../resource/) directory.

Currently:

- `kgs.yml` - YAML
- `kgs.jsonld` - JSON(LD)
- `kgs.ttl` - RDF/Turtle
- `organizations.yml` - combined organization metadata (YAML)

See the [Makefile](../Makefile) for details on how these are constructed.
