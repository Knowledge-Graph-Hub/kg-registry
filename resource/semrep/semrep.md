# SemRep

## Description

SemRep is a rule-based semantic interpreter for biomedical text developed by the
U.S. National Library of Medicine (NLM). It identifies UMLS Metathesaurus concepts
in free text via MetaMap and then uses the UMLS Semantic Network, together with
linguistic structure, to extract semantic predications: subject-relation-object
triples such as "Indomethacin INHIBITS Cyclooxygenase". Running SemRep across the
PubMed corpus produces SemMedDB, a PubMed-scale repository of these predications
that serves as an upstream source for many downstream knowledge graphs, including
NP-KG.

## Products

- **SemRep / SemMedDB SKR Tools Page** — the NLM Semantic Knowledge Representation
