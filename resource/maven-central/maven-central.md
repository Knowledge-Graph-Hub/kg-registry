## Maven Central Repository

Maven Central is the primary public repository for the Java and JVM ecosystem,
hosting package metadata, version histories, and distributable artifacts for
Maven-compatible components. Software-supply-chain knowledge graphs use Maven
Central as an upstream source of Java package identity, versions, and declared
dependencies.
