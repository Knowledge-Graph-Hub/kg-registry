# AlphaFold Protein Structure Database

The AlphaFold Protein Structure Database (AlphaFold DB) is a freely accessible
database of predicted protein structures developed by Google DeepMind and
EMBL-EBI. It provides open access to over 200 million protein structure
predictions generated by the AlphaFold deep learning system, covering the human
proteome, the proteomes of many other organisms, and the UniProt reference
proteomes.

Each entry pairs a predicted three-dimensional structure with per-residue
confidence (pLDDT) and predicted aligned error (PAE) metrics. The database is
accessible through a web interface for interactive browsing and visualization, a
REST API for programmatic access, and bulk downloads of structure files in mmCIF
and PDB formats.
