## Description

The **Vasculature Common Coordinate Framework (VCCF)** uses the human blood
vasculature as a coordinate system for locating and connecting cells throughout
the body. Rather than relying solely on anatomical partonomy, the VCCF treats
the branching tree of arteries, veins, capillaries, and microvasculature as a
network that threads between organs, mapping each vessel to the anatomical
structures it supplies or drains.

As part of the **Human Reference Atlas (HRA)**, the VCCF is implemented as the
**HRA-VCCF**, the first open, comprehensive, computer-readable database of the
adult human blood vasculature. It captures vessels and their branching
sequence, vessel types and subtypes, anastomoses, portal systems,
microvasculature, functional tissue units, vessel geometry, links to 3D
reference objects, and mappings to the regions vessels supply or drain, along
with associated cell types and biomarkers. The VCCF backs the "blood
vasculature" ASCT+B (Anatomical Structures, Cell Types, plus Biomarkers) table
and is an upstream source feeding the HRA Knowledge Graph (HRA-KG).

## Products

- **VCCF Knowledge Graph Explorer** — an interactive HRA portal view for

  biomarkers, available via the stable PURL
