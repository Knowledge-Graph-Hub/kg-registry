KB Bio 101

## Description

KB Bio 101 is a formal knowledge base that encodes the content of an
introductory college biology textbook (Campbell Biology, by Reece and
colleagues). It was developed at SRI International within the AURA knowledge
representation system as part of Project Halo, using a knowledge-factory
process in which domain experts represented each textbook sentence in logic.
The knowledge base powers the Inquire Biology prototype, an intelligent
textbook that answers questions.

The exported knowledge base is published by SRI and distributed under the
CC BY-NC-SA 3.0 license in several logic serializations, including OWL2. Its
single upstream source is the Campbell Biology textbook itself, represented
here as the `campbell-biology` source.
