## Description

The Pistoia Alliance Pharma General Ontology (PGO) is a collaborative initiative to provide a shared upper-level reference ontology for the pharmaceutical industry.

## Overview

PGO addresses the challenge of insufficient and ambiguous asset descriptions that hamper information exchange within and across pharmaceutical organizations. The project aims to serve as a "Rosetta Stone" enabling semantic interoperability by establishing agreed-upon core concepts and providing clear recommendations for controlled terminologies.

## Key Features

- **Core Concepts**: Defines essential entities most frequently used when exchanging pharmaceutical R&D data
- **Controlled Terminologies**: Specifies value-sets for key attributes using community-agreed resources
- **FAIR Principles**: Follows Findable, Accessible, Interoperable, and Reusable data management principles
- **LinkML Implementation**: Uses LinkML for representing and documenting ontology types
- **Reuse-First Approach**: Leverages existing semantic resources rather than creating new ones
- **Machine-Actionable**: Creates machine-readable documentation to facilitate automation

## Project Roadmap

**Phase 1 (2023-2025)**:
- Initial scoping and identification of core concepts
- Expert consultation and alignment
- Formalization of core concepts
- First public release using LinkML

**Phase 2 (2025-2026)**:
- Field testing and application
- Complement core-concept vocabulary
- Updated version release

## Use Cases

PGO enables:
- Unambiguous identification of essential pharmaceutical entities
- Clear declaration of semantic resources used in data exchange
- Validation and facilitation of drug development data discovery
- Promotion of interoperability across pharmaceutical organizations
- Value delivery in the drug development pipeline
