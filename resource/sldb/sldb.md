SLDB is a synthetic lethality knowledge graph and build workflow developed in the
Monarch Initiative ecosystem. The project curates synthetic lethal relationships
from published cancer studies, normalizes gene identifiers, captures assay and
perturbation context, and emits a structured interaction table that can be reused
for downstream graph construction.

The upstream SLDBGen repository includes the extraction scripts, test data, and a
tabular export of curated interactions. Records in the published table include the
paired genes, identifier mappings, perturbation types, assay information, cell
line context, and PMID-backed provenance from the source literature.
