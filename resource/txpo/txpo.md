## Description

TOXic Process Ontology (TXPO) systematizes a wide variety of terms involving toxicity courses and processes. The first version of TXPO focuses on liver toxicity.

The original ToxPilot project site (https://toxpilot.nibiohn.go.jp/) is no longer reachable. The ontology continues to be maintained on GitHub (https://github.com/txpo-ontology/TXPO) and is distributed through the OBO Foundry.
