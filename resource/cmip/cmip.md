## Description

The Coupled Model Intercomparison Project (CMIP) is a coordinated climate model
intercomparison framework organized under the World Climate Research Programme
(WCRP). CMIP defines standardized controlled vocabularies that name and describe
the models, experiments, activities, institutions, and variables used across the
project. The CMIP6 Controlled Vocabularies (CMIP6CV) are published as
machine-readable JSON and underpin the metadata used by the ESGF archives.

## Products

### CMIP6 Controlled Vocabularies

Machine-readable JSON definitions of CMIP6 identifiers (source_id, experiment_id,
activity_id, institution_id, and more).
