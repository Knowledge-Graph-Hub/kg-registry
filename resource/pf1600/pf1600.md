# pf1600 Dataset

## Overview

The pf1600 dataset offers LC-MS-based metabolomic profiles, trait annotations, and sample metadata for approximately 1,600 plant-derived food samples. Data and scripts for processing and analysis are available via GigaDB and the companion GitHub repository.
