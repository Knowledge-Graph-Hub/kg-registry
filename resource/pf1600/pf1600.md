# pf1600 Dataset

## Overview

The pf1600 dataset offers LC-MS-based metabolomic profiles, trait annotations,
and sample metadata for approximately 1,600 plant-derived food samples. DataCite
metadata for the GigaDB DOI lists the archive under CC0, references the MassIVE
raw data accession, and links the companion GitHub repository for reproducible
analysis scripts.
